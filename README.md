# Nocturne Psychoacoustic Console

A browser-based psychoacoustic synthesis and automation instrument built with
Next.js and the Web Audio API.

## Prerequisites

- Node.js `>=22.13.0`

## Quick Start

```bash
npm install
npm run dev
npm run build
```

Open [http://localhost:3000](http://localhost:3000) after starting development.

## Vercel

Import the GitHub repository in Vercel and use the **Next.js** framework preset.
The default install, build, and output settings are correct. Every push creates
a deployment; pushes to `main` update production and other branches receive
preview deployments.

## Useful Commands

- `npm run dev`: start local development
- `npm run build`: create the production Next.js build
- `npm start`: serve the production build locally
- `npm test`: build and verify the rendered application
